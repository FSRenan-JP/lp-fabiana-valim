## Design e Estética (Design System)
O visual do projeto foi meticulosamente construído para transmitir a essência Premium do serviço.
- **Cor de Fundo Principal:** Preto Premium (`#070B12`)
- **Cor de Destaque Primária:** Dourado Claro / Cobre (`#F0A13A`) e Dourado Escuro (`#D8781E`).
- **Botões de CTA:** Verde WhatsApp (`#25D366`), com animações de "pulse" suave para conversão.
- **Tipografia:** 
  - Títulos Premium: 'Playfair Display' (Serif)
  - Leitura: 'Inter' (Sans)
  - Detalhes (assinaturas/frases): 'Dancing Script'

## Stack Tecnológico
- HTML5 (Estruturação semântica, meta-tags otimizadas)
- CSS3 (Vanilla, uso de variáveis CSS para design tokens, Flexbox/Grid, responsivo sem media queries excessivas)
- JS (Vanilla, interações leves como navegação fluida)
- Hospedagem e CI/CD: **Vercel** acoplada a um repositório GitHub (`FSRenan-JP/lp-fabiana-valim`).
