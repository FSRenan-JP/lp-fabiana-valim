# Índice de Memória do Agente (MEMORY.md)

Este é o arquivo raiz do sistema de memória persistente para o projeto **Fabiane Valim - Diarista**. 
No início de cada nova sessão, o agente deve ler este arquivo e seguir os links para recuperar o contexto do projeto, histórico de ações e diretrizes estabelecidas.

## 📚 Tópicos de Contexto
Abaixo estão os documentos vitais que governam o desenvolvimento deste projeto. 

- **[Contexto do Projeto e Negócio](project-context.md)**: Detalhes sobre a cliente, paleta de cores, stack tecnológico e posicionamento de mercado. Leia para entender a essência visual e estratégica do projeto.
- **[Log Contínuo de Atividades](activity-log.md)**: O diário de bordo do projeto. Contém o histórico de todas as alterações feitas, bugs resolvidos e orquestrações concluídas. Leia para saber onde paramos na última sessão.

## 🤖 Diretrizes do Projeto
1. **Padrão de Cor**: Sempre manter a estética "Premium" (Preto, Dourado e Branco).
2. **Performance**: Como é uma LP estática de conversão rápida, não adicione bibliotecas pesadas de JS a menos que estritamente necessário.
3. **Consistência**: Manter consistência estrita de Copy (textos) e informações de contato entre a LP e o Google Meu Negócio.
